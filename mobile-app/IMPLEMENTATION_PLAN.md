# Optional Tools

### Installation Commands

```bash
# Clone and setup
git clone <repository>
cd mobile-app

# Install dependencies
npm install

# iOS specific setup (macOS only)
cd ios && pod install && cd ..

# Start development
npm run start
npm run ios    # iOS simulator
npm run android # Android emulator
```

### Environment Configuration
